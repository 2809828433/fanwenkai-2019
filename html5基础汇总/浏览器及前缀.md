## 浏览器前缀

​	使用一些css3的属性的时候应该考虑到不同浏览器的内核兼容情况，需要针对不同的内核使用不同的浏览器前缀。

| 浏览器名称 |   前缀   |
| :--------: | :------: |
|   google   | -webkit- |
|   safari   | -webkit- |
|     ie     |   -ms-   |
|  firefox   |  -moz-   |
|   opera    |   -o-    |
