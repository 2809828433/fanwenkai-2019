# iframe 标签

**iframe  用于链接其他的网址**

```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>	iframe</title>
</head>
<body>
	<iframe src="http://www.baidu.com" frameborder="0" width="400"	></iframe>
	<iframe src="http://www.taobao.com" frameborder="0"></iframe>
	<iframe src="http://www.jd.com" frameborder="0"></iframe>
</body>
</html>
```
